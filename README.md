# Streamlit ElasticNet example

Были рассмотрены линейные модели:

* обычная линейная регрессия
* Ridge
* Lasso
* ElasticNet

Использовался GridSearchCV с некоторыми наборами параметров и 10 фолдами. Датасеты вначале рассматривались без категориальных признаков, затем они были добавлены и преобработаны с помощью OneHotEncoding.

Нормализация признаков не дала прироста качества, но заметный прирост дало использование категориальных фичей. Чтобы улучшить модель, можно поработать с текстом и другой категориальной фичей, которая была дропнута.



Для запуска ввести команды:



`python -m venv venv`



`venv\\Scripts\\activate`



`pip install -r requirements.txt`



`streamlit run app.py`
